PartTool

Select the partitioning tool to use.

  • PETSc

    Use PETSc interface to various partitioning tools.

  • METIS

    Use direct interface to Metis.

GraphType

Different algorithms to make the sparse graph with weighted edges from the multidimensional mesh. Main difference is dealing with neighborings of elements of different dimension.

  • any_neighboring

    Add edge for any pair of neighboring elements.

  • any_wight_lower_dim_cuts

    Same as before and assign higher weight to cuts of lower dimension in order to make them stick to one face.

  • same_dimension_neghboring

    Add edge for any pair of neighboring elements of same dimension (bad for matrix multiply).

TimeGovernor

constructible from key: max_dt
Setting of the simulation time. (can be specific to one equation)

  • TimeGovernor::start_time

    Double:
    Default: 0.0
    Start time of the simulation.

  • TimeGovernor::end_time

    Double:
    Default: Infinite end time.
    End time of the simulation.

  • TimeGovernor::init_dt

    Double: [0, ]
    Default: 0.0
    Initial guess for the time step. Only useful for equations that use adaptive time stepping.If set to 0.0, the time step is determined in fully autonomous way if the equation supports it.

  • TimeGovernor::min_dt

    Double: [0, ]
    Default: Machine precision.
    Soft lower limit for the time step. Equation using adaptive time stepping can notsuggest smaller time step, but actual time step could be smaller in order to match prescribed input or output times.

  • TimeGovernor::max_dt

    Double: [0, ]
    Default: Whole time of the simulation if specified, infinity else.
    Hard upper limit for the time step. Actual length of the time step is also limitedby input and output times.
